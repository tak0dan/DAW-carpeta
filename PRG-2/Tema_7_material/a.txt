# **✔ Example UML-like Code Structure**

```java
public abstract class Contenido {
    protected String titulo;
    protected boolean disponible;

    public Contenido(String titulo, boolean disponible) throws AtributoVacioException {
        if (titulo.equals(""))
            throw new AtributoVacioException();
        this.titulo = titulo;
        this.disponible = disponible;
    }

    public boolean isDisponible() { return disponible; }

    public abstract double calcularDuracion();
}
```

### **Pelicula**

```java
public class Pelicula extends Contenido {
    private double duracion;
    private double duracionExtra;

    public Pelicula(String titulo, boolean disponible, double dur, double extra)
        throws AtributoVacioException {
        super(titulo, disponible);
        this.duracion = dur;
        this.duracionExtra = extra;
    }

    @Override
    public double calcularDuracion() {
        return duracion + duracionExtra;
    }
}
```

### **Podcast**

```java
public class Capitulo {
    private double duracion;
    public Capitulo(double d) { this.duracion = d; }
    public double getDuracion() { return duracion; }
}

public class Podcast extends Contenido {
    private List<Capitulo> capitulos;

    public Podcast(String titulo, boolean disponible) throws AtributoVacioException {
        super(titulo, disponible);
        this.capitulos = new ArrayList<>();
    }

    public void addCapitulo(Capitulo c) { capitulos.add(c); }

    @Override
    public double calcularDuracion() {
        double total = 0;
        for (Capitulo c : capitulos)
            total += c.getDuracion();
        return total;
    }
}
```

### **Serie**

```java
public class Temporada {
    private double duracion;

    public Temporada(double d) { this.duracion = d; }
    public double getDuracion() { return duracion; }
}

public class Serie extends Contenido {
    private List<Temporada> temporadas;

    public Serie(String titulo, boolean disponible) throws AtributoVacioException {
        super(titulo, disponible);
        temporadas = new ArrayList<>();
    }

    public void addTemporada(Temporada t) { temporadas.add(t); }
    public int numTemporadas() { return temporadas.size(); }

    @Override
    public double calcularDuracion() {
        double total = 0;
        for (Temporada t : temporadas)
            total += t.getDuracion();
        return total;
    }
}
```

## **2.1. `duracionTotalDisponible(List<Contenido> l)`**

Returns total duration of available content.

```java
public static double duracionTotalDisponible(List<Contenido> l) {
    double total = 0;
    for (Contenido c : l) {
        if (c.isDisponible())
            total += c.calcularDuracion();
    }
    return total;
}
```

We do need downcasting *here* to check number of seasons.

```java
public static void mostrarSeriesConMasDeUnaTemporada(List<Contenido> l) {
    for (Contenido c : l) {
        if (c instanceof Serie) {
            Serie s = (Serie) c;  // Downcasting allowed
            if (s.numTemporadas() > 1)
                System.out.println("Serie: " + s.titulo);
        }
    }
}
```

# **5. Number + LinkedList Exercise (2 points)**

### **Requirements:**

* LinkedList<Number>
* 10 objects: 5 Doubles, 5 Integers
* Include numbers between 4 and 8
* Traverse backwards and print
* Traverse forwards and delete numbers in [4,8]
* Show only Doubles after

---

## **Code:**

```java
public static void main(String[] args) {
    LinkedList<Number> lista = new LinkedList<>();

    // Add 10 numbers
    lista.add(2);
    lista.add(4.5);
    lista.add(8);
    lista.add(1);
    lista.add(6.7);
    lista.add(10);
    lista.add(4);
    lista.add(5.0);
    lista.add(3);
    lista.add(7);

    // Backwards traversal
    ListIterator<Number> it = lista.listIterator(lista.size());
    System.out.println("Hacia atrás:");
    while (it.hasPrevious()) {
        System.out.println(it.previous());
    }

    // Delete numbers between 4 and 8 inclusive
    it = lista.listIterator();
    while (it.hasNext()) {
        double n = it.next().doubleValue();
        if (n >= 4 && n <= 8)
            it.remove();
    }

    System.out.println("Lista tras borrar [4,8]: " + lista);

    // Show only Doubles
    System.out.println("Solo Doubles:");
    for (Number n : lista) {
        if (n instanceof Double)
            System.out.println(n);
    }
}
```
