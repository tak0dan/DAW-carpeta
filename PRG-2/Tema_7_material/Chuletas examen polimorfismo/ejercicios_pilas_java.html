Recorrer una pila sin destruirla (hacia delante)
Muestra todos los elementos de la pila usando una pila auxiliar para no perder los datos originales.
public static void recorrerPila(Stack<Integer> pila) { Stack<Integer> aux = new Stack<>(); // Vaciamos la pila original en la auxiliar while (!pila.isEmpty()) { aux.push(pila.pop()); } // Mostramos y restauramos while (!aux.isEmpty()) { int elem = aux.pop(); System.out.println(elem); pila.push(elem); } } // Uso: Stack<Integer> p = new Stack<>(); p.push(1); p.push(2); p.push(3); recorrerPila(p); // imprime: 1 2 3 // La pila sigue intacta después
La pila guarda elementos en orden LIFO (último en entrar, primero en salir). Para recorrerla de base a cima sin destruirla necesitas una pila auxiliar.
Recorrer una pila al revés (hacia atrás, de cima a base)
Muestra los elementos desde la cima hasta la base usando una pila auxiliar.
public static void recorrerInverso(Stack<Integer> pila) { Stack<Integer> aux = new Stack<>(); // Guardamos todos en auxiliar (invirte el orden) while (!pila.isEmpty()) { aux.push(pila.pop()); } // Mostramos desde la cima de aux (= base de original) // y restauramos la pila original while (!aux.isEmpty()) { int elem = aux.pop(); System.out.println(elem); // orden: cima → base pila.push(elem); } } // Uso: Stack<Integer> p = new Stack<>(); p.push(1); p.push(2); p.push(3); // cima = 3 recorrerInverso(p); // imprime: 3 2 1
Pasar todos los elementos a una pila auxiliar invierte automáticamente el orden, ya que cada pop() saca la cima y cada push() la mete de nuevo en sentido contrario.
Copiar una pila
Devuelve una copia exacta de la pila sin modificar la original.
public static Stack<Integer> copiarPila(Stack<Integer> pila) { Stack<Integer> aux = new Stack<>(); Stack<Integer> copia = new Stack<>(); // Paso 1: volcamos en aux (queda invertida) while (!pila.isEmpty()) { aux.push(pila.pop()); } // Paso 2: volcamos aux en copia y restauramos original while (!aux.isEmpty()) { int elem = aux.pop(); copia.push(elem); pila.push(elem); } return copia; } // Uso: Stack<Integer> original = new Stack<>(); original.push(10); original.push(20); original.push(30); Stack<Integer> copia = copiarPila(original); // original: [10, 20, 30] ← intacta // copia: [10, 20, 30] ← igual
Se necesitan dos pilas auxiliares: una para invertir y otra para copiar. Al final, aux se vuelca a copia y a la vez se restaura la original.
Buscar un elemento en la pila
Devuelve true si el elemento existe en la pila, sin destruirla.
public static boolean buscarEnPila(Stack<Integer> pila, int valor) { Stack<Integer> aux = new Stack<>(); boolean encontrado = false; while (!pila.isEmpty()) { int elem = pila.pop(); if (elem == valor) { encontrado = true; } aux.push(elem); } // Restaurar la pila original while (!aux.isEmpty()) { pila.push(aux.pop()); } return encontrado; } // Uso: Stack<Integer> p = new Stack<>(); p.push(5); p.push(12); p.push(8); System.out.println(buscarEnPila(p, 12)); // true System.out.println(buscarEnPila(p, 99)); // false
No hacemos break al encontrarlo para poder seguir restaurando la pila. Si quisiéramos ser más eficientes podríamos parar al encontrarlo y luego restaurar desde aux.
Contar cuántas veces aparece un elemento
Recorre la pila contando las ocurrencias de un valor dado.
public static int contarOcurrencias(Stack<Integer> pila, int valor) { Stack<Integer> aux = new Stack<>(); int contador = 0; while (!pila.isEmpty()) { int elem = pila.pop(); if (elem == valor) { contador++; } aux.push(elem); } while (!aux.isEmpty()) { pila.push(aux.pop()); } return contador; } // Uso: Stack<Integer> p = new Stack<>(); p.push(3); p.push(7); p.push(3); p.push(3); p.push(1); System.out.println(contarOcurrencias(p, 3)); // 3
El patrón es siempre el mismo: recorrer vaciando en aux, operar, y luego restaurar volcando aux de vuelta a la original.
Convertir una pila en ArrayList
Pasa todos los elementos de la pila a un ArrayList (de base a cima), sin destruir la pila.
public static ArrayList<Integer> pilaALista(Stack<Integer> pila) { Stack<Integer> aux = new Stack<>(); ArrayList<Integer> lista = new ArrayList<>(); // Volcamos en aux para invertir el orden while (!pila.isEmpty()) { aux.push(pila.pop()); } // De aux sacamos en orden base→cima while (!aux.isEmpty()) { int elem = aux.pop(); lista.add(elem); // añadimos a la lista pila.push(elem); // restauramos original } return lista; } // Uso: Stack<Integer> p = new Stack<>(); p.push(1); p.push(2); p.push(3); ArrayList<Integer> lista = pilaALista(p); System.out.println(lista); // [1, 2, 3]
Útil para combinar pilas con otros ejercicios del tema (como los de ArrayList que ya tienes). El índice 0 de la lista corresponde a la base de la pila.
Comprobar si una pila de caracteres es un palíndromo
Clásico ejercicio de pilas: comprueba si la secuencia de caracteres se lee igual al derecho que al revés.
public static boolean esPalindromo(Stack<Character> pila) { Stack<Character> aux = new Stack<>(); Stack<Character> copia = new Stack<>(); // Copiamos la pila en aux (queda invertida) while (!pila.isEmpty()) { aux.push(pila.pop()); } // Restauramos original y guardamos copia invertida while (!aux.isEmpty()) { char c = aux.pop(); pila.push(c); copia.push(c); // copia = pila al revés } // Comparamos pila original vs copia invertida Stack<Character> orig2 = new Stack<>(); boolean esPalin = true; while (!pila.isEmpty()) { char c1 = pila.pop(); char c2 = copia.pop(); if (c1 != c2) esPalin = false; orig2.push(c1); } // Restaurar pila original while (!orig2.isEmpty()) pila.push(orig2.pop()); return esPalin; } // Uso: Stack<Character> p1 = new Stack<>(); for (char c : "radar".toCharArray()) p1.push(c); System.out.println(esPalindromo(p1)); // true Stack<Character> p2 = new Stack<>(); for (char c : "hola".toCharArray()) p2.push(c); System.out.println(esPalindromo(p2)); // false
El palíndromo con pilas es un ejercicio muy típico de examen. La clave: una copia invertida de la pila original es simplemente la pila leída de cima a base.
