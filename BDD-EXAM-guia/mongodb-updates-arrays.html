03. Mutaciones

Como cambiar campos, anadir valores y trabajar con arrays

Esta pagina cubre la parte de escritura: cambiar un campo, devolverlo, anadir nuevos valores, borrar valores, renombrar campos y tocar arrays simples o arrays de objetos.

Inicio Construir consultas Aggregation y avanzado Biblioteca de recetas

Regla basica de update

db.coleccion.updateOne(
  {filtro},
  {$operador: {campo: valor}}
)

MongoDB separa claramente la parte que selecciona documentos y la parte que cambia documentos.

Operadores de update mas usados

Operador Para que sirve Ejemplo
$set Crear o cambiar un campo {$set: {precio: 15}}
$unset Eliminar un campo {$unset: {oferta: ""}}
$rename Renombrar un campo {$rename: {valoracion: "rating"}}
$inc Sumar o restar numeros {$inc: {precio: 2}}
$mul Multiplicar numeros {$mul: {precio: 1.10}}
$push Anadir al final de un array {$push: {ingredientes: "oregano"}}
$addToSet Anadir si no existe {$addToSet: {ingredientes: "oregano"}}
$pull Eliminar valores de un array {$pull: {ingredientes: "oregano"}}

Cambiar campos normales

Cambiar un campo

db.comidas.updateOne(
  {nombre: "Pizza Cuatro Quesos"},
  {$set: {precio: 15}}
)

Cambiar varios

db.comidas.updateMany(
  {categoria: "Pizza"},
  {$set: {
    disponible: true,
    destacado: true
  }}
)

Eliminar campo

db.comidas.updateMany(
  {},
  {$unset: {destacado: ""}}
)

Numeros y contadores

Incrementar

db.comidas.updateOne(
  {_id: 1},
  {$inc: {precio: 2}}
)

Decrementar

db.comidas.updateOne(
  {_id: 1},
  {$inc: {precio: -2}}
)

Multiplicar

db.comidas.updateMany(
  {categoria: "Pizza"},
  {$mul: {precio: 1.10}}
)

Arrays simples: anadir y quitar valores

Anadir uno

db.comidas.updateOne(
  {_id: 1},
  {$push: {ingredientes: "oregano"}}
)

Anadir varios

db.comidas.updateOne(
  {_id: 1},
  {$push: {
    ingredientes: {
      $each: ["oregano", "aceitunas"]
    }
  }}
)

Anadir sin duplicar

db.comidas.updateOne(
  {_id: 1},
  {$addToSet: {ingredientes: "oregano"}}
)

Quitar uno

db.comidas.updateOne(
  {_id: 1},
  {$pull: {ingredientes: "oregano"}}
)

Quitar varios

db.comidas.updateOne(
  {_id: 1},
  {$pull: {
    ingredientes: {$in: ["oregano", "aceitunas"]}
  }}
)

Quitar ultimo o primero

db.comidas.updateOne(
  {_id: 1},
  {$pop: {ingredientes: 1}}
)

Ordenar o recortar arrays al insertar

Insertar al principio

db.comidas.updateOne(
  {_id: 1},
  {$push: {
    etiquetas: {
      $each: ["premium"],
      $position: 0
    }
  }}
)

Dejar maximo 3

db.comidas.updateOne(
  {_id: 1},
  {$push: {
    etiquetas: {
      $each: ["nueva"],
      $slice: 3
    }
  }}
)

Orden interno

db.comidas.updateOne(
  {_id: 1},
  {$push: {
    stock: {
      $each: [{sede: "Aldaia", cantidad: 4}],
      $sort: {cantidad: -1}
    }
  }}
)

Campos anidados

Para cambiar un campo dentro de un objeto, usa dot notation.

db.comidas.updateOne(
  {_id: 1},
  {$set: {
    "origen.ciudad": "Milan",
    "proveedor.zona": "Sur"
  }}
)

Arrays de objetos y operadores posicionales

Primer elemento concreto

db.comidas.updateOne(
  {_id: 1},
  {$set: {"stock.0.cantidad": 20}}
)

Elemento encontrado con $

db.comidas.updateOne(
  {_id: 1, "stock.sede": "Valencia"},
  {$set: {"stock.$.cantidad": 25}}
)

Todos los elementos con $[]

db.comidas.updateMany(
  {},
  {$set: {"stock.$[].revisado": true}}
)

Solo algunos con $[elem]

db.comidas.updateMany(
  {},
  {$set: {"stock.$[elem].alerta": true}},
  {arrayFilters: [{"elem.cantidad": {$lt: 5}}]}
)

Upsert, replace y fecha de cambio

Crear si no existe

db.comidas.updateOne(
  {nombre: "Lasana"},
  {$set: {
    categoria: "Principal",
    precio: 11
  }},
  {upsert: true}
)

Reemplazar entero

db.comidas.replaceOne(
  {_id: 2},
  {
    _id: 2,
    nombre: "Pizza Vegana Verde",
    categoria: "Pizza",
    precio: 15,
    vegano: true
  }
)

Fecha de modificacion

db.comidas.updateOne(
  {_id: 1},
  {$currentDate: {fecha_mod: true}}
)
