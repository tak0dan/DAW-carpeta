db.comidas.find({

db.comidas.find({

db.comidas.find({
  stock: {
    $elemMatch: {

    }
  }

Distinct

db.comidas.distinct("categoria")

Aggregation pipeline: piensa en etapas

  1. $match: reducir el conjunto.
  2. $group: resumir o agrupar.
  3. $project: decidir que campos salen.
  4. $sort: ordenar.
  5. $limit: recortar.
db.comidas.aggregate([
  {$match: {categoria: "Pizza"}},
  {$group: {
    _id: "$vegano",
    total: {$sum: 1},
    precioMedio: {$avg: "$precio"}
  }},
  {$sort: {total: -1}}
])

Consultas de aggregation mas usadas

Contar por categoria

db.comidas.aggregate([
  {$group: {
    _id: "$categoria",
    total: {$sum: 1}
  }}
])

Media por categoria

db.comidas.aggregate([
  {$group: {
    _id: "$categoria",
    avgPrecio: {$avg: "$precio"}
  }}
])

Maximo por categoria

db.comidas.aggregate([
  {$group: {
    _id: "$categoria",
    maxPrecio: {$max: "$precio"}
  }}
])

Solo pizzas veganas

db.comidas.aggregate([
  {$match: {categoria: "Pizza", vegano: true}},
  {$group: {
    _id: "$categoria",
    total: {$sum: 1}
  }}
])

Renombrar campos de salida

db.comidas.aggregate([
  {$project: {
    _id: 0,
    plato: "$nombre",
    coste: "$precio"
  }}
])

Contar documentos

db.comidas.aggregate([
  {$match: {disponible: true}},
  {$count: "totalDisponibles"}
])

Unwind: abrir arrays

db.comidas.aggregate([
  {$unwind: "$ingredientes"},
  {$group: {
    _id: "$ingredientes",
    total: {$sum: 1}
  }},
  {$sort: {total: -1}}
])

$unwind convierte cada elemento del array en una fila temporal del pipeline.

Lookup: join de MongoDB

db.pedidos.aggregate([
  {$lookup: {
    from: "clientes",
    localField: "clienteId",
    foreignField: "_id",
    as: "cliente"
  }},
  {$unwind: "$cliente"},
  {$project: {
    _id: 0,
    pedido: "$_id",
    cliente: "$cliente.nombre",
    total: 1
  }}
])

Piensa en $lookup como "traeme los datos relacionados de otra coleccion".

AddFields y calculos

Descuento calculado

db.comidas.aggregate([
  {$addFields: {
    precioConIVA: {$multiply: ["$precio", 1.10]}
  }},
  {$project: {
    nombre: 1,
    precio: 1,
    precioConIVA: 1
  }}
])

Clasificar por precio

db.comidas.aggregate([
  {$addFields: {
    nivelPrecio: {
      $cond: [
        {$gte: ["$precio", 15]},
        "alto",
        "medio-bajo"
      ]
    }
  }}
])

Indices y explain

Crear indice simple

db.comidas.createIndex({
  categoria: 1
})

Crear indice compuesto

db.comidas.createIndex({
  categoria: 1,
  precio: -1
})

Ver indices

db.comidas.getIndexes()

Explain de una consulta

db.comidas.find({
  categoria: "Pizza",
  precio: {$lt: 20}
}).explain("executionStats")

Si una consulta avanzada se va a usar mucho, piensa si necesita indice. Si quieres saber como la esta ejecutando MongoDB, usa explain().
